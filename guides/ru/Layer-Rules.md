## Правила слоёв

Некоторые элементы в Wayland не являются окнами, а представляют собой слои. К ним относятся, например:
запускатели приложений, панели состояния или обои.

У них есть специальные правила, отличные от окон. Их синтаксис точно такой же,
но у них разные свойства и эффекты.

### Свойства

| Поле | Аргумент | Описание |
| -------------- | --------------- | --- |
| match:namespace | [RegEx] | пространство имён слоя, проверьте `hyprctl layers`. |

### Эффекты

| эффект | аргумент | описание |
| ---- | ----------- | --- |
| no_anim | [on] | Отключает анимации. |
| blur | [on] | Включает размытие для слоя. |
| blur_popups | [on] | Включает размытие для всплывающих окон. |
| ignore_alpha | [a] | Заставляет размытие игнорировать пиксели с прозрачностью `a` или ниже. `a` — число с плавающей точкой от `0` до `1`. `a = 0`, если не указано. |
| dim_around | [on] | Затемняет всё, что находится за слоем. |
| xray | [on] | Устанавливает режим xray для размытия слоя. `0` — выключено, `1` — включено, `unset` — по умолчанию. |
| animation | [style] | Позволяет установить определённый стиль анимации для этого слоя. |
| order | [n] | Устанавливает порядок относительно других слоёв. Большее значение `n` означает ближе к краю монитора. Может быть отрицательным. `n = 0`, если не указано. |
| above_lock | [0/1/2] | Если не ноль, отображает слой поверх экрана блокировки, когда сеанс заблокирован. Если установлено значение `2`, можно взаимодействовать со слоем на экране блокировки, в противном случае он будет отображаться только поверх него. |
| no_screen_share | [on] | Скрывает слой от демонстрации экрана, рисуя поверх него чёрный прямоугольник. |
