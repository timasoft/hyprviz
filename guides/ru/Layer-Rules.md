## Правила слоёв

Некоторые элементы в Wayland не являются окнами, а представляют собой слои. К ним относятся, например:
запускатели приложений, панели состояния или обои.

У них есть специальные правила, отличные от правил для окон:

```ini
layerrule = правило, пространство_имён
# или
layerrule = правило, адрес
```

где `rule` — это правило, а `namespace` — регулярное выражение для пространства имён (найти
пространства имён в `hyprctl layers`) или `address` — адрес в формате
`address:0x[hex]`.

### Правила

| правило | описание |
| ---- | ----------- |
| unset | Удаляет все ранее установленные layerRules для заданного регулярного выражения пространства имён. Обратите внимание, что оно должно совпадать _точно_. |
| noanim | Отключает анимации. |
| blur | Включает размытие для слоя. |
| blurpopups | Включает размытие для всплывающих окон. |
| ignorealpha [a] | Размытие игнорирует пиксели с прозрачностью `a` или ниже. `a` — число с плавающей точкой от `0` до `1`. `a = 0`, если не указано. |
| ignorezero | Размытие игнорирует полностью прозрачные пиксели. То же самое, что `ignorealpha 0`. |
| dimaround | Затемняет всё, что находится за слоем. |
| xray [on] | Устанавливает режим xray для размытия слоя. `0` — выключено, `1` — включено, `unset` — по умолчанию. |
| animation [style] | Позволяет установить определённый стиль анимации для этого слоя. |
| order [n] | Устанавливает порядок относительно других слоёв. Большее значение `n` означает ближе к краю монитора. Может быть отрицательным. `n = 0`, если не указано. |
| abovelock [interactable] | Отображает слой поверх экрана блокировки, когда сеанс заблокирован. Если установлено значение `true`, можно взаимодействовать со слоем на экране блокировки, в противном случае он будет отображаться только поверх него. |
| noscreenshare [on] | Скрывает слой от демонстрации экрана, рисуя поверх него чёрный прямоугольник. |
