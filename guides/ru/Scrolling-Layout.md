## Правила рабочей области

| имя | описание | тип |
| --- | --- | --- |
| direction | То же, что и scrolling:direction | str |

например

```ini
workspace = 2, layoutopt:direction:right
```

## Сообщения макета

Параметры диспетчера `layoutmsg`:

| имя | описание | параметры |
| --- | --- | --- |
| move | перемещает макет горизонтально, либо на относительные логические пиксели (`-200`, `+200`), либо на столбцы (`+col`, `-col`) | данные перемещения |
| colresize | изменяет размер текущего столбца, либо до значения, либо на относительное значение, например `0.5`, `+0.2`, `-0.2` или циклически переключает предустановленные с `+conf` или `-conf`. Также может быть `all (number)` для изменения размера всех столбцов до конкретной ширины | относительное float / относительное conf |
| fit | выполняет операцию fit на основе аргумента. Доступно: `active`, `visible`, `all`, `toend`, `tobeg` | режим fit |
| focus | перемещает фокус и центрирует макет, также выполняя циклический переход вместо перемещения на соседние мониторы. | направление |
| promote | перемещает окно в его собственный новый столбец | нет |
| swapcol | Меняет текущий столбец местами с соседним слева (`l`) или справа (`r`). Замена циклическая (например, замена первого столбца влево перемещает его в конец). | `l` или `r` |
| togglefit | Переключает focus_fit_method (center, fit) | нет |

Пример привязок клавиш для вашего конфигурационного файла Hyprland:

```
bind = $mainMod, period, layoutmsg, move +col
bind = $mainMod, comma, layoutmsg, swapcol l
```
